## 🎨 Theme & Palette System

We target a premium visual identity:

- **Default**: Slate/Zinc scale with primary indigo accents.
- **Color Format**: Use HSL color declarations to simplify opacity adjustments and variations:
  - `--primary`: `hsl(240, 5.9%, 10%)`
  - `--accent`: `hsl(263.4, 70%, 50.4%)`
  - `--background`: `hsl(0, 0%, 100%)`

## 🧱 Component Layout Rules

- **Design Philosophy**: Use responsive, mobile-first design approaches.
- **Interactive States**: Every clickable component MUST have styling for:
  - `:hover` — slight scale-up (e.g. `scale-102`) or opacity adjustment.
  - `:focus-visible` — clean focus rings for accessibility.
  - `:disabled` — grayed out options, and `cursor: not-allowed`.
